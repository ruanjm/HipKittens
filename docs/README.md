# Documentation 


## Docker

AMD [releases various Docker containers](https://hub.docker.com/u/rocm). We use the docker containers specified below:
- MI300X/MI325X: [launch_docker_mi300x.md](https://github.com/HazyResearch/HipKittens/blob/main/docs/docker/launch_docker_mi300x.md)
- MI350X/MI355X: [launch_docker_mi350x.md](https://github.com/HazyResearch/HipKittens/blob/main/docs/docker/launch_docker_mi350x.md)


## Profiling

We recommend using [rocprofiler](https://github.com/ROCm/rocprofiler) to profile your kernels during development. 

Within the Docker, we run the process shown in [profiling/](https://github.com/HazyResearch/HipKittens/blob/main/docs/profiling/README.md) to set up the profiler.
